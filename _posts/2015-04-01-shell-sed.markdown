#Shell
##变量
- shell的变量**很弱**,无需定义任何类型，
- 变量在赋值时，等号`=`两边必须不留任何`空格`，
- 变量在使用时可以使用`$`开头使用

##if条件判断
首先看代码

```sh
if [ ! -e "$website_dir" -o ! -e "$weblogin_dir" ]
then
	echo "$website_dir 不存在"
	echo "$weblogin_dir 不存在"
else
    ...
fi
```
